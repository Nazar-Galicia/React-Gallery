## Gallery

A modern image gallery web application built with React and Vite.  
This project was created as a portfolio project to demonstrate working with external APIs, component-based architecture, state management, and user interaction.

The gallery allows users to browse images, open them in a modal view, and navigate between images in a smooth and intuitive way.
